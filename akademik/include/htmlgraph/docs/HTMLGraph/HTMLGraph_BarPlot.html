$barReference = 

	
[line 511]

  To what reference should the size of the bars be calculated

Possible values are
  • local when the bars are drawn according to what percent their value is representing of
          the total values in the plot where the bar is located
 
  • global when the bars are drawn according to what percent their value is representing of the
          maximum value of all the bars of the graph
   Default is "global"

double

$label = ""], [string

		Creates one or more new HTMLGraph_Bar objects and adds them to the BarPlot object

If a single value is specified (not an array), through the returned pointer you can alter the properties of   the bar, the properties of bar's label through the bar's label object (of HTMLGraph_Text type)   and the properties of the bar's value label through the value object, (also of HTMLGraph_Text type)





Tags:





  

    return:  pointer to the newly created bar object if a single object is specified or void                   if an array of values is specified
