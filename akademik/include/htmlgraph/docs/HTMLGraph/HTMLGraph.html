Class Details



[line 37]

A PHP class to draw vertical bar graphs.

This class draws vertical graphs only by using HTML and CSS. No requirement for any external graphic library   such as GD
Simple and grouped bars can be created. You can change everything regarding the looks as the output is template   driven. On the fly, you can change the size of graph, the color of all or individual bars, you can add labels,   title, footnote or customize the CSS style of all the elements.
One of the finest features of this class is that you are not anymore restricted to specify the width and/or height   of your bar in pixels! You can also specify it in percents!
The output looks the same way in all major browsers.
See the documentation for more info.
